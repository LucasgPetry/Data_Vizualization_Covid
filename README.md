# 📊 Dashboard COVID-19 no Brasil

Este repositório apresenta um dashboard interativo desenvolvido no **Power BI**, com o objetivo de analisar e visualizar dados da COVID-19 no Brasil de forma clara, dinâmica e acessível.

## 🧠 Sobre o Projeto

O projeto foi criado para facilitar a compreensão da evolução da pandemia no Brasil por meio de visualizações intuitivas. Ele permite explorar dados relevantes como:

- Casos confirmados ao longo do tempo  
- Número de óbitos  
- Taxas de letaliade 
- Distribuição geográfica por estados    

## 🛠️ Ferramentas Utilizadas

- **Power BI** – Criação do dashboard e visualizações  
- **GitHub** – Versionamento e compartilhamento  
- **Base de dados pública** – Dados utilizados para análise. Fonte: https://covid.saude.gov.br/
